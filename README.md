# Kingdomino
Fully Implemented in Python3 with Typing

More here http://www.blueorangegames.eu/pf/kingdomino/

## Instructions
1. `python3.6 -m pip install colored --user`
2. `python3.6 game.py`
3. `python3.6 game.py filename.txt` For saving terminal inputs
